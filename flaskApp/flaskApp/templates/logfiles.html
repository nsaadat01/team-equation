Generating Procmon Log Files

Process Monitor is a Windows tool that monitors and logs file system, registry, and process/thread activity in real-time.

Download and Install Procmon

  1. Visit the Procmon download page.
  2. Download the tool and extract the files to a convenient location.

Run Procmon

  1. Open the extracted folder and double-click Procmon.exe to launch the application.
  2. Accept the license agreement if prompted.

Configure Filters (Optional)

  1. To focus on specific events, click on the "Filter" menu and select "Filter...".
  2. Add the filters you need, such as filtering by process name or path. Click "Add" after setting each filter, then "OK" to apply.

Start Logging

By default, Procmon starts capturing events as soon as it is opened. If you stopped it, click the "Capture" icon (magnifying glass) or press Ctrl+E to start capturing events.

Reproduce the Issue

Perform the actions you want to log. This might include running an application, opening files, etc.

Stop Logging

Once you've reproduced the issue, stop the capture by clicking the "Capture" icon again or pressing Ctrl+E.

Save the Log

  1. Go to "File" > "Save...".
  2. Select "Events displayed using current filter" to save only the filtered events or "All events" to save everything.
  3. Choose "Native Process Monitor Format (PML)" for detailed logs or "Comma Separated Values (CSV)" for a more readable format.
  4. Click "OK" and choose a location to save the log file.

Generating Wireshark Log Files

Wireshark is a network protocol analyzer that captures and logs network packets.

Download and Install Wireshark

  1. Visit the Wireshark download page.
  2. Download and install the appropriate version for your operating system.

Run Wireshark

  1. Launch Wireshark from the Start menu or desktop shortcut.

Select a Network Interface

On the main screen, you'll see a list of network interfaces. Select the one you want to capture traffic from (usually the active Ethernet or Wi-Fi connection).

Start Capturing

Click the blue shark fin icon or press Ctrl+E to start capturing packets on the selected interface.

Reproduce the Issue

Perform the network activity you want to log, such as browsing a website, running a network application, etc.

Stop Capturing

Once you've captured the necessary traffic, stop the capture by clicking the red square icon or pressing Ctrl+E.

Save the Capture

  1. Go to "File" > "Save As...".
  2. Choose a location and enter a name for your capture file.
  3. Ensure the file format is set to "Wireshark/tcpdump/... - pcapng" for detailed logs.
  4. Click "Save" to save the capture file.

Summary

  • Procmon logs system activities such as file access and registry changes.
  • Wireshark captures and logs network traffic.
  • These tools are powerful and can generate detailed log files that are helpful for troubleshooting and analysis. Ensure you have appropriate permissions to run these tools and capture the necessary data.
